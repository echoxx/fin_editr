===============================================

NOTE: Automated downloading from Investing.com is not yet working.

USE EXISTING DOWNLOADED FILES (RECOMMENDED FOR NOW)
---------------------------------------------------
1. Manually download IS and BS from Investing.com Pro
2. Save them in a folder (e.g., ./NewCompany/)
3. Run:

python netnet_main.py analyze TICKER --skip-download --folder ./NewCompany

Examples:
    python netnet_main.py analyze 7859 --skip-download --folder ../trilogiq
    python netnet_main.py analyze 7859 --skip-download --folder ../Toso


FULL AUTOMATION (NOT YET WORKING)

OTHER COMMANDS
--------------
# List existing analyses
python netnet_main.py list

# Test that credentials work
python netnet_main.py verify-credentials

# Debug mode (watch the browser)
python netnet_main.py analyze 7859 --exchange TYO --no-headless

# Dry run (see what would happen)
python netnet_main.py analyze 7859 --exchange TYO --dry-run


AFTER ANALYSIS
--------------
Open the workbook and fill in:
- Industry (manual research)
- Price (current stock price)
- Exchange Rate (if non-USD)

Then run autoscoring if you didn't provide --price:
python netnet_autoscore.py toso/toso.xlsx --price 879
